# Vue Pokédex


![pikachu-running](https://github.com/AndresZbz/vue-pokedex/assets/98718897/bb81e902-36e5-4fbf-98a0-1dd2cd0d72ad)

This is a simple Vue3 (using Vite tool) Pokédex web application using bootstrap for styling.

Also this app uses PokeApi for the API requests, you can check them here:

[Github](https://github.com/PokeAPI/pokeapi)
[Website](https://pokeapi.co/)

## Instalation

Run the ```bash npm install``` command to install all the node packages and then you can run the project using ```bash npm run dev```
